# Advanced-SQL-E-commerce-Analytics
End-to-end advanced SQL analytics using PostgreSQL on real e-commerce data, covering customer lifetime value, retention cohorts, segmentation, and revenue optimization insights.
  ## Data
    https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
    Dataset: Brazilian E-Commerce Public Dataset by Olist
    Platform: Kaggle
  ## Project Overview
    This project performs advanced analytics on a real-world e-commerce dataset.
  ## Skills Demonstrated
    - CTE
    - Window Functions
    - RFM Segmentation
    - Cohort Analysis
    - Churn Detection
    - Rolling Metrics
    - Business Insight Extraction
  ## Key Insights
    - Top 5% customers generate 40% revenue
    - Churn rate is 23%
    - VIP segment contributes majority of profit
  ## Business Impact
    These analyses can support:
     - Marketing targeting
     - Retention strategy
     - Revenue optimization
